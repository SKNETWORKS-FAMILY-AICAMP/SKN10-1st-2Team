## Introduction Project (프로젝트 개요)
### 프로젝트 명 : 자동차 등록 현황 데이터 시각화
- 프로젝트 내용
  
    - 전국 자동차 등록현황과 현대 기아차의 FAQ를 인터넷에서 데이터 크롤링
    - 데이터 정제 후 시각화 플랫폼
- 프로젝트 목표
  
    - 팀원들의 웹 크롤링으로 데이터 수집
    - 데이터 정제 후 sql에 입력
